Outcome 2: supplier correctness

The system must standardize supplier identities more reliably.

Pass when:

OCR-broken supplier names are cleaned

Outcome 3: tobacco classification correctness

The system must recognize cigarette brands and map them to the correct nominal account.

Pass when:

tobacco brands with mild OCR corruption still map correctly

Outcome 4: review precision

The system must reduce noisy reviews and improve actionability.

review rows include specific reasons

Outcome 5: auditability

Pass when:

key extracted fields retain raw value, normalized value, confidence, source snippet, and page number
